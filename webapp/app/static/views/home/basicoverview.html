What we wanted to do:

Find out how Americans interact with voting and $.

Why:

The landscape of presidential campaigning is changing rapidly changing.

All about money and marketing

Candidates are very much being “sold”

How is it being done, and how is it working? Who do they target and how do they respond?

3 Main Data Sets

The Federal Election Committee database of campaign contribution disclosures

The Guardian’s election results aggregated by county

The USDA's Economic Research Service‘s 2012 us Rural Atlas on county demographic

2 Joined Data sets

The U.S. Postal Service’s listing of ZIP codes

The Census Bureau’s listing of American counties

Tools

Data Cleanup

ALOT of Data cleanup

Use of Pandas / Pandas DataFrames - Python's version of SQL

County Level Analysis

Needed to find out Federal Information Processing Standard (FIPS) for every data point

Some data sets indexed by FIPS, some data sets indexed by Zip Codes

Needed to create mappings to derive FIPS code for many data points

Visualization

D3 to process the data

Normalized data in the front-end

Use of TONS of GeoJSONs to create map of USA - long rendering time

D3 is pretty amazing but there are many flaws i.e. write alot of code to get something done
